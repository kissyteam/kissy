Usage

new S.SlidingLabels(container, config)

S.SlidingLabels 构造器, 传入容器元素及配置项 , 例如


KISSY.ready(function(S) {
    new S.SlidingLabels('#info');
    // or
    S.SlidingLabels.autoRender();
});

Config

position - number | Array.<number>

单位 px, 表示在水平和垂直方向上, 相对于父元素的位置. x or [x, y], 不设置时, 取 0. 默认是 [5, 5].

offset - number

单位 px, 表示label 和 input 之间的距离. 默认为 5.

zIndex - number

label 的 z-index 值. 默认为 99.

duration - number

label 的运动速度. 默认为 0.2.

focusStyle - Object

输入框获取焦点时, label 的样式. 默认为 {}.

blurStyle - Object

输入框失去焦点时, label 的样式. 默认为 {}.
