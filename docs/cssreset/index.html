KISSY CSS Reset 的理念

  • reset 的目的不是清除浏览器的默认样式，这仅是部分工作。清除和重置是紧密不可分的。
  • reset 的目的不是让默认样式在所有浏览器下一致，而是减少默认样式有可能带来的问题。
  • reset 期望提供一套普适通用的基础样式。但没有银弹，推荐根据具体需求，裁剪和修改后再使用。

特色：适应中文环境，基于最新主流浏览器测试。
