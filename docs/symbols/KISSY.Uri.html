KISSY.Uri

Uri class for KISSY. Most of its interfaces are same with window.location.

defined in： uri.js

  • fragment {} fragment such as "#!/test/2".
  • hostname {} hostname such as "docs.kissyui.com".
  • path {} path such as "/index.htm".
  • port {} Port such as "8080"
  • query {} Query object for search string.
  • scheme {String} scheme such as "http:".
  • userInfo {} User credentials such as "yiminghe:gmail"

    Parameters

    Example
    • 
  this: "http://y/yy/z.com?t=1#v=2"

  "//foo" => "http://foo"
  "foo" => "http://y/yy/foo"
  "/foo" => "http://y/foo"
  "?foo" => "http://y/yy/z.com?foo"
  "#foo" => http://y/yy/z.com?t=1#foo"
