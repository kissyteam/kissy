这是 KISSY 的 DOM 模块，包含以下子模块：

  • selector CSS 选择器，提供 get, query 等方法
  • dom-data 扩展属性操作, 提供 data, removeData 等方法
  • dom-class class 属性相关操作，提供 addClass, toggleClass 等方法
  • dom-attr 通用属性操作，提供 attr, text 等方法
  • dom-style 样式操作，提供 css, addStyleSheet 等方法
  • dom-offset 位置操作，提供 offset, scrollLeft, docHeight 等方法
  • dom-traversal 遍历操作，提供 children, next, contains 等方法
  • dom-create 创建操作，提供 create, html 等方法
  • dom-insertion 插入操作，提供 insertBefore, insertAfter 等方法
