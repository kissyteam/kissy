## Vision

小巧灵活，简洁实用，使用起来让人感觉愉悦。

    Keep It
        Simple & Stupid, Short & Sweet, Slim & Sexy...
    Yeah!


## Structure

 - bin:           kissy-nodejs 命令行执行文件
 - reports:       nodejs 下的单元测试报告
 - build:         构建好的发布文件
 - docs:          API 文档
 - src:           源码、测试等开发资源
 - tools:         打包压缩等自动化工具
