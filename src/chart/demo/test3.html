使用

使用很简单

var mychart = new KISSY.Chart("#canvasid")
mychart.render(jsondata);

为了兼容IE9以下浏览器，可在页面里面引用flashcanvas，来支持Canvas what is flashcanvas? 

<!--[if lt IE 9]>
<script src="../chart/flashcanvas/flashcanvas.js"></script>
<![endif]-->
如果是IE9，由于添加了版本判断，请确保IE9以标准的文档模式渲染 
<meta http-equiv="X-UA-Compatible" content="IE=edge" $gt;

配置JSON

 
{
    //图表类型，条形图line 或 折线图
    type : "{line|bar}",
    //图表数据{array}
    elements : [{
        name : "接待人数",//{string} 数据名称
        data : [0,10,00,10,30,10], //{array} 数据序列
        label : "{d}%", // {array|string} 显示数据格式，或具体值
        format : "0",//{string} 显示数据格式[A]
        drawbg : false//{boolean} 可选
        notdraw : false //{boolean}
    }],
    //图表坐标{object}
    axis : {
        x : {
            name : "时间",//{string} 横坐标名（可选）
            labels : [], //{array} 刻度名称，与element[].data 对应
        },
        y : {
            name : "人数",//{string} 纵坐标名（可选）
            labels : [],//{array} 刻度名称，与element[].data 对应（可选）
            max : 50//{number}定义Y最高轴的数值（可选）
        }
    },
    //config (可选)
    config : {
        left : 40,//{number}左边距
        top : 38//{number}上边距
    }
}

[A]使用数据格式format

*           0 - (123456) show only digits, no precision
*           0.00 - (123456.78) show only digits, 2 precision
*           0.0000 - (123456.7890) show only digits, 4 precision
*           0,000 - (123,456) show comma and digits, no precision
*           0,000.00 - (123,456.78) show comma and digits, 2 precision
*           0,0.00 - (123,456.78) shortcut method, show comma and digits, 2 precision
