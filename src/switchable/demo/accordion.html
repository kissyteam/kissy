键盘快捷键

  • 左/上键:当焦点在标签时转到上一个标签
  • 右/下键:当焦点在标签时转到下一个标签
  • Home: 当焦点在标签时转到第一个标签
  • End: 当焦点在标签时转到最后一个标签
  • enter/space: 当焦点在当前标签时，展开对应的面板
  • Control + PgUp : 当然焦点在容器内时转到当前标签上一个标签
  • Control + PgDn : 当然焦点在容器内时转到当前标签下一个标签

Note

chrome 不支持 Control + PgUp 以及 Control + PgDn
